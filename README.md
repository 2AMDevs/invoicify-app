## âŒ¨ï¸ Development

- Clone this repo or create your own fork and then clone.
- Create your own `.env`, `.electron-builder.yml`

```bash
#.env
GH_TOKEN=<REPO TOKEN>
REACT_APP_API_URL=<API_URL_USED_FOR_OTP>
```
